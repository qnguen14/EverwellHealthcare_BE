Testing Meeting URL

To test a meeting URL like http://localhost:5173/meeting/8db052ff-7acb-4333-9071-d03abac121cf:

  1. Make sure your frontend application is running at http://localhost:5173
  2. The frontend should have the Agora SDK integrated
  3. The frontend should handle URLs in the format /meeting/{appointmentId}

If you don't have a frontend yet, the meeting URL won't work when clicked.
